# Smart IoT Dashboard

## 📌 Project Description
This project is a Smart Light Control System using ESP32/ESP8266 and a web dashboard.

It allows users to control lights and monitor sensor data like LDR (light sensor) and PIR (motion sensor).

---

## ⚙️ Features
- AUTO / MANUAL mode
- Motion detection
- Light level monitoring
- Remote control (ON / OFF)

---

## 🛠️ Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- ESP32 / ESP8266 (MicroPython)

---

## 🚀 How to Run
1. Upload code to ESP device  
2. Run Flask server (or deploy on Render)  
3. Open browser and access dashboard  

---

## 📷 Output
Smart dashboard with buttons and sensor values display
